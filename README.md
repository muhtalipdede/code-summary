# Code Summary

This extension calculates and displays the line count, function count, or character count of the code block selected by the user.

## Installation

1. Search for "Code Summary" in the VS Code [Marketplace](https://marketplace.visualstudio.com/)
2. Click the "Install" button and install the extension.

## Usage

1. Open a file in VS Code or select a code block in an already open file.
2. Run the command "Code Summary: Summarize Code" (or type "Ctrl+Shift+P" and type "Code Summary: Summarize Code").
3. Choose the option you want to use among the ones presented to you.

This extension is designed to make it easy for the user to understand and navigate large code blocks. Feel free to suggest any improvements or features that you would like to see in the future.
